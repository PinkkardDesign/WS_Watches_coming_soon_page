# WS Watches – Coming Soon Landing Page

A premium "Coming Soon" landing page developed for WS Watches to build anticipation for the launch of their first collection.

## Features

- Responsive design for desktop, tablet, and mobile
- Custom titanium-inspired branding and logo
- Netlify Forms integration for waitlist signups
- Custom favicon
- SEO-optimized metadata
- Video background with optimized mobile experience
- HTTPS secured through Netlify

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Netlify
- GitHub
